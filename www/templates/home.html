Bienvenue sur Pogopi

Pourquoi cette API :

J'ai voulu avoir une api simple, et me permettant de réaliser un mini-jeu. Jouant parfois à Pokémon Go, je me suis dit que ce genre de données se prêtaient aussi bien pour la réalisation d'un site (recherche, tri, CRUD) que d'un mini-jeu. Cependant, je n'ai pas trouvé d'API consommable ni open-source sur ce thème. pokeapi.co est certe consommable mais les données concernent l'ensemble des jeux pokémons depuis leur création, et les données ne sont pas ordonnées de manière à simplifier la récupération de données (tableau de langue dont pour un même index on obtient une langue différente, nécessite plusieurs requètes pour rassembler les données utiles, etc ...).

J'ai donc décidé d'utiliser l'api de pokeapi.co ainsi que des sites web sur Pokemon Go, pour construire une API open-source, simple et suffisament complète pour couvrir plusieurs cas d'utilisation.

Pourquoi ce site :

Le site n'est qu'un exemple d'usage de l'api, et permet également de plus facilement en voir le contenu.

Il permet également de donner plus de lisibilité à la documentation, et de fournir des exemples de codes concrêts.

La version de l'API, c'est :

  • 649 Pokémons (jusqu'à la génération 5, comme disponible sur Pokémon Go)
  • 278 chaines d'évolutions de pokémon.
  • 233 Abilités (ou talents)
  • 131 Attaques Chargées
  • 65 Attaques Rapides
  • 26 Pokéballs
  • 18 Types
  • 7 générations
  • 3 équipes disponibles dans Pokémon Go
  • 2 langues (Fr et En)
